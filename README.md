# Simple Todo App

A minimal todo application built with **HTML**, **CSS**, and **Python**.  
This project is intentionally simple and is meant to demonstrate basic full-stack fundamentals: form handling, server-side logic, and basic styling.

---

## Features

- Add new todos
- View a list of todos
- Delete todos
- Persistent storage using a local file (no database)

---

## Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Storage:** Plain text / JSON file
